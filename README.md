# 概览

云手机产品主要提供云手机服务器、云手机、手机应用、手机镜像的生命周期管理功能。同时提供Web页面和API调用两种操作方式，用户可批量申购、释放服务器，批量创建、重启云手机，批量上传、安装或卸载云手机应用，对自制的云手机镜像进行批量克隆，产品支持安卓、iOS和H5多种访问方式
